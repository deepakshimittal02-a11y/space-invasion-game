# 🚀 Space Invasion Game

A classic Space Invaders-style arcade game built with Python and Pygame.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Pygame](https://img.shields.io/badge/Pygame-2.5.2-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 🎮 Gameplay
- Control a rocket ship and shoot down 8 alien enemies
- Enemies move across screen and descend each time they hit a wall
- Game ends if any enemy reaches the bottom of the screen
- Score increases with every successful hit

## 🕹️ Controls
| Key | Action |
|-----|--------|
| ← Arrow | Move Left |
| → Arrow | Move Right |
| Spacebar | Shoot Bullet |

## ⚙️ How to Run

1. Clone the repository
```bash
   git clone https://github.com/TERA-USERNAME/space-invasion-game.git
   cd space-invasion-game
```
2. Install dependencies
```bash
   pip install -r requirements.txt
```
3. Run the game
```bash
   python game.py
```

## 🛠️ Tech Stack
- Python 3
- Pygame 2.5.2
- Pygame Mixer (audio)
